# FLAX
***Formal Languages and Automata eXplorer*** - образовательный проект, направленный на разработку актуального программного пакета для моделирования конечных автоматов и формальными языками. Проект вдохновлен **JFLAP** и должен реализовывать его функционал, но в рамках современных средств разработки - `Tauri + Rust`.

В данный момент стоит задача разработать _MVP_, реализующая *DFA*, *NFA* и *CF*-грамматики. Дальнейшая разработка проекта будет проводиться в течении 2026-2027 гг. и затронет интеграцию функционала прямо в крейт `automata`. К концу жизненного цикла проекта должно быть воспроизведено хотя бы 80% функционала JFLAP и разработана комфортная среда для работы с конечными автоматами и формальными языками.
